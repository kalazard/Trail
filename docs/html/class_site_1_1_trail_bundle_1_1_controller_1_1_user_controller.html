createAction ( )

Fonction de création d'un utilisateur

Cette méthode est appelée en ajax et requiert les paramètres suivants :

email : Email de l'utilisateur à créer nom : Nom de l'utilisateur à créer prenom : Prénom de l'utilisateur à créer datenaissance : Date de naissance de l'utilisateur à créer telephone : Téléphone de l'utilisateur à créer licence : Url du site de la licence de l'utilisateur à créer

Exemple d'appel de la méthode côté client :

$.ajax({ type: "POST", url: "{{path('site_trail_createUser')}}", data: data, cache: false, success: function (data) { if (!data.serverError) { if (data.success) { data.message = message de succès } else { data.message = message d'erreur }

} else { data.message = erreur du côté serveur }

} });

Returns
string

JSON permettant de définir si l'utilisateur a été créé ou non

Example en cas de succès :

{ "success": true, "serverError": false, "message": "Message" }

Example en cas d'erreur dans la création :

{ "success": false, "serverError": false, "message": "Message" }

Example en cas d'erreur du serveur :

{ "success": false, "serverError": true, "message": "Message" }
